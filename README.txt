
COFFEE CAFE - FULL HTML + JAVASCRIPT VERSION

1. Extract this ZIP file.
2. Open the folder in VS Code.
3. Open index.html in browser.
4. No XAMPP and no PHP are needed.
5. Register a new account using register.html.
6. Login using login.html.
7. Add drinks from menu.html.
8. Open cart.html to update quantity/remove items.
9. Checkout and place order.
10. Receipt will appear at receipt.html.
11. Analytics will update at analytics.html.

IMPORTANT:
- Data is saved in the browser using localStorage.
- This means users, cart, orders and receipt are stored only in your browser.
- For GitHub Pages, upload all files/folders exactly as they are.
- Make sure index.html is in the main/root folder.
