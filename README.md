# CoffeeCafe

# a. Project Title & Description

## Project Title

Coffee Cafe Web Application

## Project Description

Coffee Cafe Web Application is a simple café ordering system developed using HTML, CSS, and JavaScript. The system allows users to browse café menus, add items to the cart, register and log into the system, make orders, and view receipts. The application also includes a simple analytics dashboard to display sales-related information. This project was converted from PHP into a fully static web application without requiring a database or server setup.


# b. Features Included

* User Registration
* User Login Authentication
* Café Menu Display
* Add to Cart Function
* Update Cart Quantity
* Remove Cart Items
* Checkout System
* Receipt Generation
* Sales Analytics Dashboard
* Responsive Navigation Menu
* Local Storage Data Saving


# c. Instructions to Test Login

1. Open the project folder.
2. Double click `index.html` or run using VS Code Live Server.
3. Navigate to the **Register** page.
4. Create a new user account using a username and password.
5. After registration, go to the **Login** page.
6. Enter the registered username and password.
7. Successful login will redirect the user into the system.
